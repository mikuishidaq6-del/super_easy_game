# 血液がん患者向け育成ゲームアプリ

**Concept:** 来ただけで成長するゲーム\
体調が悪い日でも負担なく使える、血液がん患者をそっと支えるアプリ。

------------------------------------------------------------------------

# アプリ概要

本アプリは、血液がん患者が **短時間・低負担で毎日使える**
ことを目的とした育成ゲーム型アプリである。

ユーザーはログイン時に **5段階のフェイススケール**
で体調を入力するだけでよい。\
入力内容に応じてキャラクターが反応し、経験値が蓄積され、キャラクターが成長する。

また、歩数・セルフケア・服薬記録などの生活行動もポイントとして反映される。

**体調が悪い日でも10秒で完了する設計**を重視する。

------------------------------------------------------------------------

# コア体験（基本フロー）

    アプリ起動
       ↓
    ホーム画面
       ↓
    フェイス入力（今日の体調）
       ↓
    キャラクター反応
       ↓
    経験値加算・成長

この流れは **約10秒で完了する設計**とする。

------------------------------------------------------------------------

# 実装機能

## 1. フェイス入力

ログイン時に体調を5段階で入力する。

**質問**

    きょうの調子は？

**入力方法**

-   5段階フェイススケール
-   1タップ入力
-   文章入力なし

**目的**

-   体調が悪い日でも入力可能
-   毎日続けられるシンプル設計

------------------------------------------------------------------------

## 2. キャラクター反応

フェイス入力に応じてキャラクターが反応する。

例：

  フェイス   キャラコメント
  ---------- ----------------------------
  😊         いい日だね
  🙂         今日も来てくれてありがとう
  😐         来ただけでOK
  😟         無理しないでね
  😢         今日は休む日でも大丈夫

表示要素

-   キャラクター表情
-   一言メッセージ

------------------------------------------------------------------------

## 3. 成長システム

フェイス入力により **経験値（EXP）** が付与される。

### 成長要素

-   経験値
-   レベル
-   コイン

### 連続ログイン

ログイン継続日数に応じてボーナスが付与される。

### 救済機能

    15コイン → 連続ログイン切れを回復

------------------------------------------------------------------------

## 4. 歩数連動

歩数に応じてコインを獲得できる。

### 基本ルール

    100歩 = コイン獲得

### 体調補正

フェイススケールに応じて歩数価値を変更する。

例：

  フェイス   歩数価値
  ---------- ------------
  良い       通常
  普通       少し増加
  悪い       大きく増加

※体調が悪い日でも達成感を得られる設計

------------------------------------------------------------------------

## 5. アイテム・食べ物

コインを使ってアイテムを購入できる。

### アイテム用途

-   キャラクターにプレゼント
-   キャラクターリアクション

例

    リンゴ
    スープ
    ぬいぐるみ
    花

------------------------------------------------------------------------

## 6. セルフケア記録

日常ケアを記録できる。

### 記録項目

-   歯磨き
-   うがい（最大5回）
-   体拭き
-   シャワー

### 目的

-   入院生活のセルフケア支援
-   行動の可視化

実施するとポイントが付与される。

------------------------------------------------------------------------

## 7. 服薬記録

服薬状況を簡単に記録できる。

### 入力方法

    飲めた
    飲めなかった

### 特徴

-   ワンタップ入力
-   詳細入力なし
