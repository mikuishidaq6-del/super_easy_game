#　コイン取得計算

## 定義
プレイヤーが1にちあたりに獲得可能なコイン取得量を定義します

### 通常時のコイン獲得関数

Coin(x) = L / 1 + e^(-k(x - x0))

- L: コイン獲得の上限 (= 20)
- k: 成長の速さを調整するパラメータ (= 0.01)
- x0: コイン獲得の成長が最も速くなる歩数(目標歩数) (= 500)
- x: プレイヤーの歩数

- e: ネイピア数（約2.71828）

### 体調不良時のコイン獲得関数
Coin(x) = a * ln(x + 1)
- a: 体調不良時のコイン獲得のスケーリングパラメータ (= 2.8)
- ln: 自然対数関数 (底がeの対数関数)
- x: プレイヤーの歩数

### (sup) リハビリ用のコイン獲得関数

- 閾値以上歩くと獲得可能コインが減る関数

Coin(x) = L * exp(-(x - μ)^2 / (2 * σ^2))
- L: コイン獲得の上限(= 20)
- μ: コイン獲得のピークとなる歩数(= 1200)
- σ: コイン獲得の広がりを調整するパラメータ (= 500)
- exp: 指数関数
- x: プレイヤーの歩数
